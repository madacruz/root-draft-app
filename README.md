# Root Draft App

Petite application Streamlit pour tirer au sort et drafter les factions de Root.

## Stack

- Python
- Streamlit
- Google Sheets via gspread

## Fonctionnalités prévues

- Création d'une partie
- Tirage de 5 factions
- Première faction obligatoirement Militant
- Tirage de l'ordre du tour
- Choix de la map par le premier joueur
- Choix optionnel du deck par le deuxième joueur
- Draft des factions en ordre inverse
- État partagé via Google Sheets
